CSS = cascading stylesheet language

Add the style tag/element in the head of the document. In side the style element is CSS code. CSS code contains a selector (WHAT is being styled) and style rules (HOW it looks). If using a tag/element name, just the tag name with curly brackets following. Then the attribute followed by a colon, then the style selection, end with a semicolon.


Re font-family, user must have the font noted or will not display as expected (will user default font on user's browser). So, font-family allows multiple options. Note that Brackets gives drop-down hints. First items are NOT specific fonts, but CATEGORIES of fonts; browser will select the default cursive/monospace/serif etc. font the user has. Or, can choose specific font family. So, best to use a groups of fonts.

To apply same style item to various elements, list each element separated by a comma

Font size can be changed by POINT (more for print design). PIXELS, or EMs. One 1 (1em) is standard font size of user's browser. Can use increments to enlarge (1.5em, 2em, 2.5 em, etc.) ir decrease (.75em, .5 em, etc.) font size.

The font-variant option allows inherit, normal, and small-caps.

Can set LEADING (how closely/loose/y set the lines are. Use line-height. The em-scale works the same as for font-size.

The text-decoration lets us set various properties such as underline and overline. Don't generally use underline because it makes it seem like the text is a line. But, sometimes overline can give a good effect.

There are LOTS more of these style options besides those shown here and even within the options of each style item.

To style LISTS in CSS, previously learned how to put the TYPE attribute in the ul or ol tag, but can also handle this in CSS. Use list-style-type. Again, MANY options within this to look at.

Can use an image as a bullet. list-style-image. Need to have image created ahead. Also need to have the image file in the same folder as the html document. (or else have to use a path in the file name) Example =
list-style-image: url(bullet.png);

To adjust the indentation value in lists by using the padding-left: attribute. This goes insdie the li CSS. To control the placement of the list markers relative to the edge of the list, use list-style-postion:; options are inside or outside. This goes inside the ol/ul CSS.


CSS Classes and IDs. Any content can be marked with a CLASS or ID to use with style tage (also can be used by JavaScript).
For ID, add the id="idname" in the opening tag of the element to be styled. Then, in the CSS, add that id prefaced by the # sound. Each id on the page is unique, i.e., can only be assigned to one element.
Classes can be assigned to muliple elements in a document. In CSS, they are prefaced by a period. Can use class="" inside a paragraph tag, but more often used with SPAN tags.

Spans can be used with classes and ids can be used with any element.

LINKS and MEDIA

hyperlinks

external links: hyperlinks from current site/domain to external site/domain

  • First part of link is the test the user will select; this is surrounded by anchor tags.
  • In the opening anchor tag, put the href="" with the fully-qualified url of the link inside the quotes
  • sample: You can take this course on Udemy.
  • if you link to a site that doesn't exist or the site changes its address = broken link

internal links: link to page within your site/domain

  • href in internal links is only the file name OR
  • This is a link to the restaurant review project.
  • Should NOT have deadends (where user needs to use back button) in internal links, so create a return [internal link] from the linked page back to the original page.
