The title in the head has 3 functions

  • The name of the page TAB in the browser
  • Used by Google for indexing
  • If user bookmarks the page, this is the name that will show.

The title in the head does NOT show on the browser page.

Break tags can be used within paragraphs to break a line. The break tag
is an empty tag - not an opening and closing tag, just the tag with the slash included.

Titles on the page are accomplished via the heading tabs (h1-h6).

h1 is the most important heading

h2 is next in importance

h3 is next

this is an example to h4

h5 follows
h6 is the least important

Note that CSS can be used to further style each of these heading tags. Default is larger and bold, but CSS easily overrides the default.

Lists are either ORDERED ol or UNORDERED ul

Change the appearance of the ORDERED list by adding the TYPE attribute in the opening ol tag. "A" creates capital letter ordering, "a" lower case eltters, 1 = regular numbers, I = uppercase Roman numerals, i= lowercase Roman numerals.

For UNORDERED lists, type can be disc (default-so not needed), circle, square

Places I'd like to visit

  1. Kenya
  2. Tanzania
  3. Someplace quiet

This display an unordered (bulleted) list
