Tables. Tables in essence are grids. First tags are table (table) and row. Within the tr tag, have either table heading row (th) or table data row (td). Tables also have columns.
Separate th elements within the same tr in essence create the number of columns.
Do NOT need to put on separate lines in code, but can for ease of human reading the code.
Can put a border attribute in the table tag (NOT recommended); use CSS instead.
The rowspan tag goes in the first (highest) td tag to be combined into one table unit.
The colspan tag goes in the left-most td tag to be combined into one tag.
