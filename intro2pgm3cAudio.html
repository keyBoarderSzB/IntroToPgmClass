Some Audio Info

Initial tag is audio, attribute control="controls" uses the controls of the users browser, which will vary some from browser to browser.

Different browsers may support different audio file types (wav, mp3, ogg, etc.). Most browsers nowadays support mp3.

NOTE: If there is a space in the file name, it will be represented by a %20 in the linked file name.

For file types not supported on a browser, list a second source. If browser does not support the first audio file type, will look at the next one.
